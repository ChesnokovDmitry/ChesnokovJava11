Задание 4.
Дана скобочная последовательность: [((())()(())]]
Можно ли считать эту последовательность правильной?

Ответ - нет. Чтобы последовательность стала правильной, необходимо добавить одну квадратную скобку и одну круглую.
Возможные варианты:
[[((()))()(())]]
[((()))[()(())]]
[((())())[(())]]
[[((())()(()))]]
